## 📖 使用说明

1. **笔记**：按学科/语言找到对应目录，直接往里丢 Markdown
2. **刷题**：`exercises/leetcode/easy/` 里每道题一个文件，格式参考 `templates/`
3. **面经**：`interview/experiences/` 按校招/社招/实习分类，写完后记得脱敏
4. **造轮子**：`projects/mini/` 里的项目别看名字带 tiny，面试时能吹半小时
5. **收藏资料**：`resources/` 里丢链接+简短评价，防止"收藏 = 学会"的错觉
