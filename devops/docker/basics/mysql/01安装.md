# 拉取 MySQL 镜像
* 在终端执行以下命令，建议拉取 8.0 版本：
```bash
sudo docker pull mysql:8.0
```

# 创建数据持久化目录
* 为了保证数据在容器删除后不丢失，需要在主机创建挂载目录：
```bash
mkdir -p /opt/mysql/data
```


# 运行 MySQL 容器
* 执行以下命令启动 MySQL，请将 your_password 替换为您自己的密码：
```bash
sudo docker run -d \
  --name mysql-container \
  -p 3306:3306 \
  -v /opt/mysql/data:/var/lib/mysql \
  -e MYSQL_ROOT_PASSWORD=78120900 \
  mysql:8.0
````

* -d: 后台运行
* -p 3306:3306: 端口映射（主机:容器）
* -v: 数据卷挂载（持久化数据）
* -e MYSQL_ROOT_PASSWORD: 设置root密码

# 验证安装

* 查看容器是否运行成功：
```bash
sudo docker ps
```


# 进入 MySQL 交互界面

* 如果需要进入容器使用 mysql 命令行工具：
```bash
sudo docker exec -it mysql-container mysql -uroot -p
```
